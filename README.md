# phpexcel_recalculate

> This is a example to show how PHPExcel recalulate after changing cell value.

## Install

### 1. Download Composer

```
wget https://getcomposer.org/composer.phar
```

### 2. Install PHPExcel

```
php composer.phar install
```

### 3. Run script

```
php run.php
```
